# Industry Financial Data Visualization

Comparative financial analysis of 10 publicly listed companies in the PC, server, and hardware sector, using web-scraped financial data.

## Analysis
1. **Data Collection** - Web scraping of stock codes and financial metrics from public databases
2. **Profitability Comparison** - Line charts comparing operating profit margin, ROE, ROA, and turnover ratios across 10 companies
3. **Industry Benchmarking** - Radar charts comparing top performers (Hikvision, Dahua) against industry best-in-class metrics
4. **DuPont Analysis** - Decomposition of ROE into component ratios with trend visualization
5. **Rose Charts** - Nightingale rose charts for multi-year profitability trends
6. **Quarterly Breakdown** - Donut charts and bar charts for seasonal revenue/profit patterns
7. **Solvency & Cost Analysis** - Debt ratios, current/quick ratios, and cost structure visualization

## Visualizations
- Line charts, radar charts, rose (Nightingale) charts
- Donut charts, grouped bar charts
- Industry benchmark comparison plots
